Application Overview

Start to Finish is a running application that helps people who want to enahnce thier running perfornaces. The user enters in their personal bests in all the events that they have run and then they enter in their time goals in the event that they are wanting to improve in. Then the application gives you a basic training plan for your user to follow to best achieve that goal.

Technology Overview

HTML, CSS, JavaScript Python for backend GPS tracking from phone Basic data analysis for performance trends Hosted on GitHub Pages or Render
